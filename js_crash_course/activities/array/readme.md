# Javascript Functions and Array

```
Create a simple page, that will show the even numbers and odd numbers separately based on the inputs from the user in developer's console using javascript prompt.

The page will accept multiple inputs and stops when the user enters 0.

NO LOOP STATEMENT ALLOWED

```

# Example
```
let Inputs = 2,3,7,4,8,10,21,14

Result:

Even numbers : 2,4,8,10,14
Odd numbers : 3,7,21
```
